# Gyms insertion

**Make sure you've generated the `places.json` and `zones.json` files inside the `data/` folder before running the bulk script**

## Instructions

1. Install packages:

```bash
pnpm i
# npm i
```

2. Create `.env` file with the `MONGO_URI`.

3. Run `bulk` script:

```bash
pnpm bulk
# npm run bulk
```
